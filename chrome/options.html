Record management

If you want to keep your records permanently, please save the JSON files somewhere permanent for yourself! Don't rely on browser storage being permanent. The JSON file can generate SVGs, but the SVG is not sufficient to generate the JSON, so if you want accurate record-keeping, make sure to get the JSON file and not just the SVG.
